# 创作决策记录（Decision Log） - 避免重陷歧途

## 决策 01｜时间线调整
**决定：** 设定为 **初三**，而非成年后或初一。
**原因：**
- 成年后太沉重，带有责任感，不够纯粹。
- 初一太幼小，缺乏对恋爱的自觉渴望。
- 初三正好处于“渴望恋爱但经验为零”的青涩期，且面临升学，更有故事张力。

## 决策 02｜人设重塑（素世）
**决定：** 采用 **“软糯/腼腆”** 素世，而非“妈妈/成熟”素世。
**原因：**
- 打破读者惯性思维，制造反差萌。
- 实现与祥子恋爱关系的“对等性”，避免变成单纯的照顾关系。

## 决策 03｜人设重塑（祥子）
**决定：** 采用 **“白祥/理想主义”**，强调其与爱音的区别。
**原因：**
- 祥子的主动源于“对美的追求”和“书本理论”，爱音的主动源于“市侩/实用主义”。
- 必须通过“贵族气质”和“生活天然呆”来确立祥子的独特性。

## 决策 04｜剧情驱动模式
**决定：** **内部驱动**（角色性格导致），而非外部驱动（巧合/狗血事件）。
**原因：**
- 想要天然的甜，必须让角色自己想靠近。
- 例如：去废弃车站不是因为迷路，而是因为素世宠溺祥子的浪漫幻想。

## 决策 05｜叙事视角
**决定：** **无评语的第三人称上帝视角（隐形摄影机）**。
**原因：**
- 增强沉浸感，不打断读者的体验。
- 能够公平地展现两人的内心戏（双向暗恋）。
