# 人物理解备忘 - 防止严重OOC

## 长崎素世 (Soyo)
- **核心台词风格**：温吞、留有余地、句尾柔和、偶尔因害羞而结巴。
- **行为逻辑**：
    - "我想做，但我怕给你添麻烦/怕你不喜欢" -> 需要祥子推一把。
    - "虽然我不懂你的浪漫，但我愿意陪你疯" -> 极致的宠溺。
- **雷区**：不要写成八面玲珑的社交达人，那是她的伪装，在祥子面前她只是个恋爱新手。

## 丰川祥子 (Sakiko)
- **核心台词风格**：典雅、书面语、偶尔带点大小姐腔调 (desuwa)、引用书本/诗歌。
- **行为逻辑**：
    - "书上说这是恋人该做的，所以我们要执行" -> 理论派。
    - "这个好美/好有趣，我要给素世看" -> 纯粹的分享欲。
    - "日常生活好难，素世救救我" -> 无意识的依赖。
- **雷区**：不要写成爱音那种自来熟的元气感。她的热情是内敛而专注的。
